--- layout: default breadcrumbs: [['/program','programs']] pagetitle: Program: webscore --- The webscore program identifies measure numbers in SCORE data. Without options, the program will generate a list of identified measure numbers. The l will label the measure numbers with @function: barnum parameters. Using m will also mark the associated barline with the measure number.

OPTIONS

ⓘ-i Include item index serial numbers.
ⓘ-r Print replacement expressions.
ⓘ-s Page index number for first system on extracted page.
ⓘ-p Indicate part number in class tags.
ⓘ-R Do not round quarter-note timestamps.
ⓘ-A Do not label articulations.
ⓘ-a Abbreviated markup. Embed only note ids into font 99 text. Use -r option to create a side-channel markup file.
ⓘ--aa Markup all items, not just notes.
ⓘ--sep string Separator between filename and system enumerator.

EXAMPLES

    Example webscores generated with this program:
