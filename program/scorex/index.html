--- layout: default breadcrumbs: [['/program','programs']] pagetitle: Program: scorex --- The scorex program extracts page data from PPMX files.

OPTIONS

ⓘ-c Count the number if input pages/segments.
ⓘ-i Display information about input pages, overlays, and systems.
ⓘ-p Extract the given indexed page (offset from 1)
ⓘ--pc Count the number of input pages only.
ⓘ--sc Count the number of input segments only.
