--- layout: default title: Programs xpersistent: program pagetitle: Utility Programs --- This pages lists command-line utility programs that use the Scorelib library. The source code for these programs can be found on GitHub.

barlinenum Identify measure numbers in SCORE data.
courtesy Identify courtesy accidentals (accidentals which do not alter the pitch of the note).
fingernum Identify fingering numbers in SCORE data.
pagenum Identify page numbers in SCORE data.
score2mei Convert data from SCORE to MEI.
score2musicxml Convert data from SCORE to MusicXML.
scorecat Converts binary data to PMX format. Assembles multiple files into a single PPMX file.
scorex Extract page data from PPMX files.
