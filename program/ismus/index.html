--- layout: default breadcrumbs: [['/program','programs']] pagetitle: Program: ismusnum --- The ismusnum program identifies binary SCORE files. SCORE binary files usually end in ".mus", but they can also be found with other extensions, such as ".pag" for explicitly indicating a full page of music, or ".pt" to indicate a instrumental part.

OPTIONS

ⓘ-b List only bad files (files which are not binary SCORE files).
ⓘ-g List only good files (files which are binary SCORE files).
ⓘ-r Recursively search all input directories for binary SCORE files.
