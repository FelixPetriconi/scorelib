--- layout: default breadcrumbs: [['/program','programs']] pagetitle: Program: scorecat grouptitle: ['https://github.com/craigsapp/scorelib/blob/master/src-programs', 'Source Code'] group: [ ['https://github.com/craigsapp/scorelib/blob/master/src-programs/scorecat.cpp', scorecat.cpp], ] --- The scorecat program can be used to combine multiple pages into a single multi-page stream. The program can also time-sort the items on each page by system as well as mark barlines and calculate the durations of measures.

OPTIONS

ⓘ-b Mark barlines in data.
ⓘ-c Encode multipage info as comments
ⓘ-s Sort items by system.
ⓘ-z Index barlines and system numbers from zero instead of one.
