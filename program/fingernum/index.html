--- layout: default breadcrumbs: [['/program','programs']] pagetitle: Program: fingernum --- The fingernum program identifies finger numbers in SCORE data. Without options, the program will generate a list of identified finger numbers. The l will label the finger numbers with @function: fingernum parameters. Using m will also mark the associated barline with the finger number.

OPTIONS

ⓘ-l Label page number items with a @function: barnum parameter.
ⓘ-A Don't analyze finger numbers before further processing (for example when removing manually marked finger numbers).
ⓘ-r Remove fingerings from input data.
ⓘ-R Remove everything except fingerings from the data (preserving formatting with invisible staves and durations).
ⓘ--text-only Only consider text items for finger number candidates (don't consider number items).
ⓘ--numbers-only Only consider number items for finger number candidates (don't consider text items).
ⓘ--above-cutoff 20.0 Only search for finger number below the given vertical position.
