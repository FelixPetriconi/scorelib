--- layout: default breadcrumbs: [['/program','programs']] pagetitle: Program: score2mei --- The score2mei program convert SCORE data into MEI data.

OPTIONS

ⓘ-f filebase Basename of file for multiple segments.
ⓘ-s # Process the given segmentation number.
ⓘ-D Use default options for Dufay translations.
ⓘ-I Don't convert invisible rests.
ⓘ-L Don't print comment with location of each measure in score.
ⓘ-M Treat entire input as a single movement.
ⓘ--debug Print debugging information.
ⓘ--no-lyrics Don't process lyrics.
ⓘ--no-page-breaks Don't print page break information.
ⓘ--no-system-breaks Don't print system break information.
ⓘ--scale # Global rhythmic scaling of durations.
